[[isalpha]]
isalpha
-------

[source,c]
---------------------
#include <ctype.h>

int isalpha (char c);
---------------------

DESCRIPTION::

The +isalpha+ function tests +c+ to determine if it is an alphabetic character (`'A'-'Z'` or `'a'-'z'`).

RETURN VALUE::

The +isalpha+ function returns a value of 1 if +c+ is an alphabetic character and a value of 0 if it is
not.


SEE ALSO::

+isalnum+, +iscntrl+, +isdigit+, +isgraph+, +islower+, +isprint+, +ispunct+, +isspace+, +isupper+, +isxdigit+

EXAMPLE::

[source,c]
---------------------
include::src/isalpha.c[]
---------------------

OUTPUT::

.....
$ gcc -Wall isalpha.c
$ ./a.out 
isalpha (?) NO
isalpha (@) NO
isalpha (A) YES
isalpha (B) YES
.....
